Platforms and Project Types

Monogame projects are targeted at a platform, for example Windows, Mac, and Linux desktops. In order for monogame to run on these platforms, certain libraries are required. For example, running Monogame on a Windows machine may require DirectX, or OpenGL, or a library/SDK specific to that version of Windows. In order to develop for certain platforms, you may be required to download and install additional software. To view available project templates that target a variety of platforms, launch a supported IDE with Monogame installed.







Monogame Windows Desktop Project

A Monogame project for Windows using DirectX.



Monogame Cross Platform Desktop Project

A Monogame project for Windows, Mac, and Linux using OpenGL.



Monogame Windows 10 Project

A Monogame project for Windows 10 UWP. Requires Windows 10.



Monogame Android Project

A Monogame project for Android. Requires Mono for Android by Xamarin.



Monogame iOS Project

A Monogame project for iOS. Requires Xamarin.iOS.



Monogame OUYA Project

A Monogame project for OUYA. Requires Xamarin.Android 4.8.2 or greater.



Monogame Content Pipeline Extension Project

A Monogame project for creating a custom content importer and processor.
